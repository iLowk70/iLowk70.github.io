Preparation

Before start, you need to boot an usb with arch linux Here. Once you have it, the next step is to enter the installer.

Basic configuration

Load your keys

For your convenience, we will change the keyboard settings to work correctly with your keyboard.

1

loadkeys <lenguage>

To see all available languages

1

ls /usr/share/kbd/keymaps/**/*.map.gz

The output should be something like this

Time and Date

Now we are going to set the date and time. For this we have a tool called timedatectl which main function is to query or modify the time and date.

1

timedatectl set-ntp true

And for synchronise

1

timedatectl status
